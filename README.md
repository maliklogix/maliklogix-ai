## ⚡ Core Features & Integrations

### 🤖 Intelligent Chatbot (Mistral AI)
- A highly-interactive, floating Chatbot engineered with **Framer Motion** physics (bouncing, scaling, talking GIF effects).
- Powered directly by the **Mistral AI API** (`mistral-tiny`).
- Custom System Persona: Programmed as an "Online Strategy Expert" created by **Malik Logix**, capable of answering complex digital marketing and SEO queries.

### ✉️ Direct AJAX Contact Form (Formspree)
- Eliminates the friction of opening external mail clients (`mailto:`).
- Uses a secure `fetch` POST request to **Formspree** for direct inbox delivery.
- Features dynamic loading states ("Sending..."), animated spinners, and a bespoke success UI component upon completion.

### 🌀 Advanced UI/UX & Animations
- **Smooth Scrolling**: Integrated **Lenis** for a cinematic, buttery-smooth scroll experience.
- **GSAP Squeeze Navbar**: A complex navbar that reacts to scroll velocity and direction, squeezing and expanding dynamically.
- **Three.js Neural Network**: An interactive 3D particle background in the Hero section that reacts to cursor movement (`@react-three/fiber` & `@react-three/drei`).
- **Intersection Observers**: Staggered fade-ins, timeline reveals, and physical perspective hover cards utilizing GSAP ScrollTrigger.
- **Context-Aware Navigation**: A highly-animated "Back to Top" component that reveals itself dynamically and initiates a smooth scroll-to-top momentum.

### 📄 Multi-Page Architecture
- Built with **React Router DOM** for instantaneous page transitions.
- **Home (`/`)**: Core acquisition funnel with case studies, timelines, feature matrices, and problem agitation.
- **Services (`/services`)**: In-depth breakdowns of AI Ads, SEO, and Analytics layers.
- **Philosophy (`/philosophy`)**: Manifestos, quotes, and the agency's overarching strategy.
- **Contact (`/contact`)**: High-conversion intake lead generation form.

## 🛠 Tech Stack Details

| Layer | Technology | Purpose |
| :--- | :--- | :--- |
| **Frontend Core** | [React 18](https://reactjs.org/) | Component-based UI architecture |
| **Build & Tooling** | [Vite](https://vitejs.dev/) | Ultra-fast development and optimized bundling |
| **Styling** | [Tailwind CSS](https://tailwindcss.com/) | Modern, utility-first CSS framework |
| **Micro-Interactions**| [Framer Motion](https://www.framer.com/motion/) | Chatbot physics, presence animations, and layout transitions |
| **Scroll Animation** | [GSAP](https://greensock.com/gsap/) | Professional-grade timeline and scroll-triggered animations |
| **3D Rendering** | [Three.js](https://threejs.org/) | Interactive 3D graphics and particle arrays |
| **Form Backend** | [Formspree](https://formspree.io/) | Serverless form endpoint submission |
| **AI Processing** | [Mistral AI](https://mistral.ai/) | LLM backend for the dynamic growth strategist chatbot |

## 🚀 Quick Start

```powershell
# Clone the repository
git clone https://github.com/maliklogix/ai-digital-marketing.git

# Enter the directory
cd ai-digital-marketing

# Install dependencies (React, Three.js, GSAP, Framer Motion, Lenis)
npm install

# Start development server on port 5176
npm run dev
```
